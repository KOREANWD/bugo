🚀 Netlify 배포 방법

  1. 위 항목을 모두 입력한 뒤 ⬇ HTML 내보내기 클릭 → index.html 다운로드
  2. netlify.com/drop 접속 (무료, 회원가입 불필요)
  3. 다운로드된 index.html을 페이지 중앙에 드래그앤드롭
  4. 자동으로 URL 생성 → 카카오톡 등으로 공유

※ 내용 수정 시 내보내기 → Netlify에 다시 업로드하면 URL은 그대로 유지됩니다.
